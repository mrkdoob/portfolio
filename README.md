# React Portfolio using NQ React Boilerplate

Work in progress!

The goal is to showcase some projects and features without showing entire repos.

[Check it out](https://marks-portfolio.netlify.com)

### Includes:

- TypeScript
- Parcel (zero config bundler)
- React Hooks
- Custom theme

### Install:

```sh
$ yarn
```

### Develop:

```sh
$ yarn start
```

### Build

```sh
$ yarn build
```
